# Etymology ── プロジェクト名の由来

## 思兼神（思金神／オモイカネ）

- 出典: 『古事記』『日本書紀』
- 参照: [古事記学センター 神名データベース](https://kojiki.kokugakuin.ac.jp/shinmei/)
- 父: 高御産巣日神（タカミムスヒノカミ）
- 主たる関係神: 天照大御神

### 神話における役割

1. **天の石屋戸（あまのいわやと）の段** ── 天照が岩戸に隠れた際、「思金神に思はしめて」八百万神の祭祀を計画。隠れた光をどう取り戻すかという根源的な「不可逆の問題」を設計で解いた神。
2. **葦原中国平定・天孫降臨の段** ── 高御産巣日神と天照の諮問に応じて、誰を派遣すべきかを評議し献策した。
3. 『日本書紀』では「深く謀り遠く慮ひ」「思慮の智有り」と評される。
4. 単なる知識神ではなく **「祭祀を実行するための政務を取り計らう」統治機能** を持つ。

### なぜ Mind Uploading OS の名前にふさわしいか

| 思兼神の働き | OmoikaneOS が解くべき課題 |
|---|---|
| 失われた光（天照）を取り戻す祭祀の設計 | 失われゆく自我（人間の心）を保存・再構成する基盤の設計 |
| 八百万神を統べて役割を割り当てる | 無数の AI サブエージェントを統率しタスクを分配する |
| 諮問に応じて計画を進言する | ユーザの意図を受けて長期計画を立案する |
| 政務を取り計らう | システム自身の自己統治・自己改修 |

「演算」ではなく「**取り計らい**」がカーネルの本質である。これがプロジェクト名の核。

### 名乗り規則

- 正式名: **OmoikaneOS**
- カーネル中枢: **Omoikane Council**（評議体）
- サブエージェント群: **Yaoyorozu**（八百万）
- ユーザの自我側: **Amaterasu Endpoint**（天照端点）
- 実行を司る Builder 群（Codex 等）: **Ame-no-Uzume**（天宇受売 ── 実際に祭祀を演じた神）
