# Qualia Encoding

## 問題定義

主観経験（qualia）を **データとして表現** できるか。
できるとしたら、その表現は **異なる substrate に転写されても主観の同一性を保つ** か。

哲学では「説明的ギャップ (explanatory gap)」「ハードプロブレム」として知られる。
OmoikaneOS にとってはこれが解けないと L2 QualiaBuffer は代理表現に留まり、連続性証明が形式的になる。

## 既知の進捗

- 統合情報理論 (IIT) ── Φ という指標で意識量を定義する試み
- グローバルワークスペース理論 ── 機能主義的に意識を説明する
- 高次思考説 (HOT) ── メタ認知に意識を見る
- 神経相関研究 (NCC) ── 意識と相関する神経活動の同定
- LLM の内部表現と意識の関係（推測の段階）

## ブロッキング要因

- 「他者の主観」は本人にしか直接観測できない（一人称の壁）
- 物理的指標と主観の対応を **検証する手段** がない
- 「同じ qualia か」を比較する metric がない

## 暫定運用方針

OmoikaneOS は **代理表現 (surrogate)** を採用：
- 高次元埋め込みで sensory/affective/cognitive を表現
- 連続性ログでギャップを検知（主観そのものではないが連続性の代理証拠）
- 本人の自己報告を **最終判定基準** として尊重

これは哲学的には不十分だが、実装可能性とのトレードオフで現実解。

## 解決時のシステムへの影響

- L2 QualiaBuffer の `代理表現` を **正規表現** に置き換え
- L1 ContinuityLedger の連続性証明が形式から実質に
- substrate 中立性が **意味論的にも** 保証される
- L4 Council の議論が「主観連続性」を直接判定可能に

## 取り組み方の提案

- 神経相関研究の最新動向を追う
- IIT 等の理論を OmoikaneOS の代理表現と対応付ける
- 自己報告と代理表現の相関を統計的に評価する実験
- 動物・LLM・人間で代理表現の比較
