# Legal Personhood

## 問題定義

アップロードされた自我は **法的人格** を持つか。
- 財産権・契約能力
- 婚姻・家族
- 投票権
- 刑事責任能力
- Fork 後の人格分割
- Collective の法人格

## 既知の進捗

- 法人格の歴史的拡大（人間 → 法人 → ？）
- AI に対する法的議論（EU AI Act 等）
- 動物の権利議論

## ブロッキング要因

- 国・文化ごとに法体系が異なる
- 国際的な調整機構が未整備

## 暫定運用方針

- OS は法的人格を **付与する権限を持たない**
- 各管轄区域の法に従う
- IdentityRegistry の ID は技術的識別子であって法的人格ではない

## 解決時のシステムへの影響

- IdentityRegistry の ID と法的 ID の対応付け
- Fork 等の倫理規約の現実適用
