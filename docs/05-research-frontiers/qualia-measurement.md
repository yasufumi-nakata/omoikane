# Qualia Measurement

## 問題定義

主観時間の連続性を **物理量として測定** できるか。
連続性の客観的証拠を、自己申告以外で取得する方法はあるか。

## 既知の進捗

- EEG での意識状態指標
- fMRI による意識相関
- Perturbational Complexity Index (PCI; Massimini ら)
- 麻酔・睡眠・植物状態での意識検出研究

## ブロッキング要因

- 客観指標と主観経験の対応が **検証不能**

## 暫定運用方針

- 自己申告 + 代理表現の相関を連続性証拠として運用
- 客観指標が確立するまでは **「複数指標の合致」** を準連続性証拠とする

## 解決時のシステムへの影響

- ContinuityLedger.qualia-checkpoint の実証性が向上
- Identity Confirmation Test の客観化
