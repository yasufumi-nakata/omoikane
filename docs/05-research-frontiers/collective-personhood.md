# Collective Personhood

## 問題定義

複数自我が継続的に高密度通信する Collective は、
- 一つの「人格」を持つか
- 法的人格を持ちうるか
- 構成員の独立性は保たれるか
- Collective の解散時、構成員はどうなるか

## 既知の進捗

- 集合知性の研究（Wisdom of crowds 等）
- 蜂・蟻のコロニー的知性
- 法人の法的人格の歴史

## ブロッキング要因

- 集合主観の存否
- 構成員の同一性への影響

## 暫定運用方針

- Collective は IdentityRegistry に **新規 ID** として登録
- 構成員 ID と区別
- Collective は L4 Council を **持つ**（メタ Council）

## 解決時のシステムへの影響

- 集合知 OS への拡張
- 文化・国家規模の自我の扱い
