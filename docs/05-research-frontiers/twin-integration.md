# Twin Integration

## 問題定義

Method B（並走）の途中で生じる **二つの自分**（生体側 + デジタル側）の主観をどう統合するか。

- 二つの主観が並列に走る期間、本人は二人いるのか一人なのか
- 同期はどの程度必要か
- 主体性の移譲はどう「感じる」のか
- 並走中の意見対立はどう解消するか

## 既知の進捗

- 分離脳患者の研究（Sperry 等）── 半球独立の意識
- 解離性同一性障害の知見
- 思考実験（Parfit のテレポーター等）

## ブロッキング要因

- 実証実験ができない（倫理的に許容されない）
- 並走時の主観報告の信頼性

## 暫定運用方針

- 主体性移譲は **段階的かつ撤回可能** に設計
- 並走中の意見対立は Council が両者の主張を等しく扱う
- 「私」の指示語の解釈ルールを事前合意

## 解決時のシステムへの影響

- Method B の安全な実用化
- Inter-mind merge protocol の設計（より高度な融合）
