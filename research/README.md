# Research ── 人間 yasufumi のノート

人間 yasufumi（または同等の研究者）が **自由に書き散らす** 場所。
形式は問わない。仮説・実験ログ・文献メモ・寝起きのアイデア・グラフ・写真も可。

## 構造

```
research/
  notes/         # 自由記述
  experiments/   # 実験計画と結果
  references/    # 文献メモ
```

## docs/ との関係

- research/ の知見が成熟したら docs/05-research-frontiers/ に整理する
- 設計判断には docs/ を真とする
- research/ は **草稿置き場**

## 命名

任意。日付プレフィックス推奨：`YYYY-MM-DD_<topic>.md`

## 機械可読性

不要。人間用。
ただし将来 Researcher Agent が読む可能性は念頭に。
